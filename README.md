# IPSSI-site-front

##Installation :

  `git clone git@github.com:Dev6-CKS/IPSSI-site-front.git
  cd IPSSI-site-front`

  Webpack installatiion:
  `npm install gulp -g`

  Install node modules:
  `npm install`

  Run project:
  `gulp`

  Open the following link in your browser:
  `http://localhost:8000`
